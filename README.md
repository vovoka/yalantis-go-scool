# Minimal counter of users curretly connected to the server.

Simple example based on [node js](https://nodejs.org/en/).
#### Requrements:
* socket.io
* express
* http

#### To run:
* git clone
* `node index.js`
* Open `http://localhost:3000/` x times to se how the counter changes it's value in all tabs.

No error handling.
